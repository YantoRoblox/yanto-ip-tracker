# Yanto IP Tracker

Tool sederhana untuk melacak lokasi IP di Termux menggunakan API publik (ipapi.co & ip-api.com).

## Fitur
- Cek IP dan lokasi sendiri
- Track IP target
- Tampilkan link Google Maps langsung
- Tampilan keren dengan banner ASCII
